What is Pose AI Processor?

Pose AI Processor is a lightweight, modular desktop application designed for real-time video pose estimation and intelligent posture analysis. Built with a clean 10-file architecture, it separates concerns for maximum maintainability, fault isolation, and easy customization.

The system processes video frame-by-frame using Google's MediaPipe library to detect and render human skeletal poses. It then optionally sends the extracted pose data to a local AI model (running in LM Studio) for detailed posture analysis, recommendations, and health insights.

Zero cloud dependencies. Zero API costs. Complete privacy.

10-File Architecture

Clean separation of concerns for maximum maintainability

📄 01_main.py

Entry point. Initializes logger, loads configs, and starts the UI.

⚙️ 02_constants.py

Central location for all hardcoded values: paths, URLs, default settings.

📝 03_logger.py

File-based logging system. No terminal needed. Safe debugging.

💾 04_config_manager.py

Manages JSON configs, LM settings (temperature, top_k, etc.), and user preferences.

🧠 05_pose_engine.py

MediaPipe Pose detection. CPU-optimized. Frame → landmarks conversion.

🎥 06_video_processor.py

Video I/O, frame-by-frame processing, background threading, progress tracking.

📊 07_json_handler.py

Generates unique JSON files with pose data, metadata, and AI configuration.

🤖 08_ai_client.py

Communicates with LM Studio. Sends pose data for AI analysis with custom settings.

🎨 09_ui_components.py

Reusable CustomTkinter widgets: styled buttons, progress bars, config panels.

🖥️ 10_ui_main.py

Main GUI window. Orchestrates all components and connects UI to backend.

Installation & Setup

Get up and running in minutes

1Clone the Repository

Download the source code from GitHub:

git clone https://github.com/HJKMCS/pose-ai-processor.git
cd pose-ai-processor

2Create Virtual Environment

Set up an isolated Python environment:

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

3Install Dependencies

Install all required Python packages:

pip install -r requirements.txt

4Setup LM Studio

Download and install LM Studio, then:

  • Load your preferred AI model (e.g., Llama 3, Mistral)
  • Go to the Local Server tab
  • Click "Start Server" (default port: 1234)

5Run the Application

Launch the Pose AI Processor:

python 01_main.py

System Requirements

Software Requirements

  • Python 3.8 or higher
  • LM Studio (for AI features)
  • Git (optional, for cloning)
  • FFmpeg (optional, for video codecs on Linux/Mac)

Hardware Requirements

  • CPU: Any modern processor (Intel/AMD)
  • RAM: 4GB minimum (8GB recommended)
  • Storage: 500MB for app + model space
  • GPU: Not required (CPU-only processing)

Python Dependencies

  • customtkinter (GUI framework)
  • opencv-python (video processing)
  • mediapipe (pose detection)
  • numpy (numerical operations)
  • requests (HTTP client for LM Studio)

How It Works

A simple, powerful workflow

📂

1. Select Video

Choose any video file (MP4, AVI, MOV, MKV) containing a person. The app supports multiple formats and resolutions.

🎥

2. Process Video

The app reads the video frame-by-frame, detects human poses using MediaPipe, and draws skeletal overlays on each frame.

💾

3. Save Results

A new video with pose overlays is saved, along with a unique JSON file containing all pose data, landmarks, and metadata.

🤖

4. AI Analysis

Optionally send the pose data to your local AI model for intelligent posture assessment, recommendations, and health insights.
